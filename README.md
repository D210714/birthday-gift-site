# 🎉 Birthday Gift Website

A fun and interactive birthday website created using HTML and CSS.

## ✨ Features

- Personalized birthday greeting
- Circular profile picture
- Beautiful gradient background
- Gift box hover effects
- GIF animations revealed on hover
- Responsive and clean design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Happy Monkey)
- GIF animations from Giphy

## 📸 Preview

The website displays:
- Birthday person's profile photo
- Age and birth date
- Multiple gift sections
- Surprise GIFs when hovering over gift boxes
